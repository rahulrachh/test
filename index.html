Height Estimation

This Project determines the height of a person from photo/video

Flowchart of how the Project works

  • Take an image
  • Convert to Grayscale
  • Rezing to 400x400
  • Use HOG descripter and SVM
  • Padding
  • Transform image if tilt
  • Compare with reference height
  • Output
